# virtuekeeper-web

Static marketing and legal site for **VirtueKeeper** — a mobile app for
practicing Benjamin Franklin's Thirteen Virtues system.

Served at [virtuekeeper.com](https://virtuekeeper.com) via GitHub Pages.

## Contents

| File           | URL                                      |
|----------------|------------------------------------------|
| `index.html`   | `https://virtuekeeper.com/`              |
| `privacy.html` | `https://virtuekeeper.com/privacy.html`  |
| `terms.html`   | `https://virtuekeeper.com/terms.html`    |
| `support.html` | `https://virtuekeeper.com/support.html`  |
| `style.css`    | shared styles (parchment palette)        |
| `CNAME`        | custom domain declaration                |

## Deployment

Push to `main`. GitHub Pages rebuilds automatically.
